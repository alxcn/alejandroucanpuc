Motivation

An industrial engineer is studying a production system with 4 workstations (A, B, C, D) whose outputs depend on shared resources (raw materials, labor hours). The system’s behavior can be modeled using vector spaces, where:
• Each workstation’s output is a dimension in a vector space.
• Resource constraints define subspaces of feasible production vectors.
We want to understand the space of possible output combinations based on the next two constraints:
1. The outputs of workstation A and B are equal to the outputs C and D.
2. The output of workstation D needs the same labor as two times the output of workstation A.

Describe the vector space of possible outputs, find a basis and based on the basis describe how the workstations contribute to the overall production system.

Linear combinations

From the previous session, we know that for a given vector space \(V\) every addition of two or more vector is also a vector in \(V\). We can extend this idea to the addition of multiple vectors scaled by some coefficients.

Definition: Let \(V\) be a vector space. A linear combination of vectors \(v_1, v_2, \ldots, v_n\) in \(V\) is an expression of the form: \[ c_1 v_1 + c_2 v_2 + \ldots + c_n v_n \] where \(c_1, c_2, \ldots, c_n\) are scalars (coefficients).

For example, in the vector space \(\mathbb{R}^2\), consider the vectors \(v_1 = (1, 0)\) and \(v_2 = (0, 1)\). A linear combination of these vectors could be: \[3v_1 + 4v_2 = 3(1, 0) + 4(0, 1) = (3, 4).\] In the vector space of matrices \(M_{2x2}(\mathbb{R})\), consider the matrices \(A = \begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix}\) and \(B = \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}\). A linear combination of these matrices could be: \[2A - B = 2\begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix} - \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix} = \begin{pmatrix} 2 & -1 \\ -1 & 2 \end{pmatrix}.\]

One question to arises is: given a set of vectors in a vector space and a target vector, can we express the target vector as a linear combination of the given vectors? The answer is it depends, and we will explore this question.

Example 1: In the vector space \(\mathbb{R}^3\), consider the vector \(v_1=(1,2,0),\) \(v_2=(0,1,1)\) and \(v_3=(-2,0,3)\). Determine if the vector \(u=(1,3,3)\) can be expressed as a linear combination of \(v_1, v_2,\) and \(v_3\). If so, find the coefficients.

Solution: We want to find scalars \(c_1, c_2,\) and \(c_3\) such that \[u = c_1 v_1 + c_2 v_2 + c_3 v_3.\] This leads to the system of linear equations \[\begin{align*} 1 &= c_1 - 2c_3 \\ 3 &= 2c_1 + c_2 \\ 3 &= c_2 + 3c_3 \end{align*}\] Solving this system, we find that \(c_1 = 1, c_2 = 1,\) and \(c_3 = 0\). Therefore, \(u\) can be expressed as a linear combination of \(v_1, v_2,\) and \(v_3\) as follows: \[u = 1v_1 + 1v_2 + 0v_3.\]

Example 2: In the vector space \(P_2(\mathbb{R})\) of all polynomials of degree at most 2, consider the polynomials \(p_1(x) = x^2 + 1,\) \(p_2(x) = x + 1,\) and \(p_3(x) = 2x^2 - x.\) Determine if the polynomial \(q(x) = 3x^2 + 2x + 1\) can be expressed as a linear combination of \(p_1, p_2,\) and \(p_3\). If so, find the coefficients.

Solution: We want to find scalars \(c_1, c_2,\) and \(c_3\) such that \[q(x) = c_1 p_1(x) + c_2 p_2(x) + c_3 p_3(x).\] This leads to the system of equations based on the coefficients of the polynomials: \[\begin{align*} 3 &= c_1 + 2c_3 \\ 2 &= c_2 - c_3 \\ 1 &= c_1 + c_2 \end{align*}\] Solving this system, we find that \(c_1 = 1, c_2 = 2,\) and \(c_3 = 1\). Therefore, \(q(x)\) can be expressed as a linear combination of \(p_1, p_2,\) and \(p_3\) as follows: \[q(x) = 1p_1(x) + 2p_2(x) + 1p_3(x).\]

Generated Spaces

Given a list of vectors, can we describe the minimal vector space that contains them? The answer is yes, and this leads us to the concept of generated spaces.

Definition: Given a set of vectors \(S\) in a vector space \(V\), we can consider the set of all possible linear combinations of those vectors. This set forms a new vector space called the span or generated space of the original set of vectors and it is denoted by \(\text{span}(S)\) or \(\langle S \rangle\). Formally, \[\text{span}(S) = \{ c_1 v_1 + c_2 v_2 + \ldots + c_n v_n \mid v_i \in S, c_i \in \mathbb{R}\}.\]

For example, in the vector space \(\mathbb{R}^2\), consider the vectors \(v_1 = (1, 0)\) and \(v_2 = (0, 1)\). The span of these vectors is the entire vector space \(\mathbb{R}^2\), since any vector \((x, y)\) in \(\mathbb{R}^2\) can be expressed as a linear combination of \(v_1\) and \(v_2\): \[ (x, y) = x(1, 0) + y(0, 1).\] In the vector space of polynomials \(P_2(\mathbb{R})\), consider the polynomials \(p_1(x) = x^2\) and \(p_2(x) = x\). The span of these polynomials is the set of all polynomials of degree at most 2 that can be expressed as linear combinations of \(p_1\) and \(p_2\): \[ \text{span}\{p_1, p_2\} = \{ c_1 x^2 + c_2 x \mid c_1, c_2 \in \mathbb{R} \},\] which includes all polynomials of the form \(ax^2 + bx\) where \(a\) and \(b\) are real numbers.

Example 3: In the vector space \(\mathbb{R}^3,\) consider the vectors \(v_1=(2,0,0),\) \(v_2=(1,3,0)\) and \(v_3=(1,1,0)\). Describe the span of these vectors.

Solution: The span of the vectors \(v_1, v_2,\) and \(v_3\) is the set of all linear combinations of these vectors. Any vector in the span can be expressed as: \[c_1 v_1 + c_2 v_2 + c_3 v_3 = c_1(2,0,0) + c_2(1,3,0) + c_3(1,1,0) = (2c_1 + c_2 + c_3, 3c_2 + c_3, 0).\] This means that the span of these vectors is the set of all vectors in \(\mathbb{R}^3\) where the third component is zero. Therefore, \[\text{span}\{v_1, v_2, v_3\} = \{(x,y,0) \mid x,y \in \mathbb{R}\},\] which is a subspace of \(\mathbb{R}^3\).

Linear Independence and Dependence

Sometimes a set of vectors can be redundant, meaning that some vectors can be expressed as linear combinations of others. For instance, in the previous example, we can see that \(v_1\) can be expressed as a linear combination of \(v_2\) and \(v_3\) as follows: \[v_1 = 2v_2 - v_3.\] This leads us to the concepts of linear independence and dependence. (Corregir)

Definition: A set of vectors is said to be linearly independent if any linear combination of these vectors that equals the zero vector all its coefficients equal to zero. In other words, if the only solution to the equation \[c_1 v_1 + c_2 v_2 + \ldots + c_n v_n = 0\] is \(c_1 = c_2 = \ldots = c_n = 0\), then the set of vectors is linearly independent. Conversely, if there exists a non-trivial solution (at least one coefficient is non-zero) to the equation above, then the set of vectors is linearly dependent.
